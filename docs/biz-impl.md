# 业务流程要求
--- 反向引用: 见 [设计文档](design.md) ---

1. 用户输入场景提示词和示例图片, 由controller调用agent编排调用不同的agent完成任务, 
2. agent运行的过程中, 需要有一个状态展示图, 展示出大模型的调用情况, 以及不同agent的调用情况, 以及执行结果、状态切换等内容
3. 系统输出一张按照用户要求产生的图片, 以及生成图片的算法代码, 以及算法的评估结果(包括系统性的数据评估, 以及多模态大模型生成的评语)
4. 用户可以对生成的图片和算法代码进行反馈, 选择Accept或 Reject with suggestions, controller接受任务后会继续根据用户的需求去改进算法, 进行下一轮的迭代优化, 直到用户满意为止
